--Iniciar el proyecto:
Paso 1:
Traer proyecto:
```
git clone https://github.com/etec-integration-project/etec-pi-2024-frontend-Santiii4.git
```
```
cd etec-pi-2024-frontend-Santiii4
```
Paso 2:
Instalar node:
```
npm install
```
Paso 3:

Iniciar con:
```
npm start
```
en la consola de visual studio.
